개요

트위터에 근무하던 마크 오토와 제이콥 손튼이 개발한 UI 라이브러리.
모바일 환경에 적합한 반응형 웹을 쉽게 개발하고, 디자인으로 인해 길어지는 개발시간을 단축하기 위해 만들었음.

getbootstrap.com (영문)
bootstrapk.com (한글)

설치방법 : bootstrap CSS 파일은 head에 link태그를 이용해서 등록,
bootstrap JS 파일은 body에 script태그를 이용해서 등록하면 됨.
물론 bootstrap 관련 외부 라이브러리(jQuary)도 등록해두어야 함
